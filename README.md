# Fang Cheng Academic Homepage

This is a lightweight static academic homepage for Fang Cheng. It can be opened directly in a browser or deployed with GitHub Pages.

## Edit Content

- Main page content: `index.html`
- Visual style: `styles.css`
- Navigation highlight behavior: `script.js`
- Profile photo: `profile.jpg`

The page includes academic profile information, research themes, publications, education, and contact details.

## Preview Locally

Open `index.html` in a browser.

## Deploy With GitHub Pages

1. Create a GitHub repository named `<username>.github.io`.
2. Put these files in the repository root.
3. Push to GitHub.
4. Open `https://<username>.github.io/` after GitHub Pages finishes publishing.
